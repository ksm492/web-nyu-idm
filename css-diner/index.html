Learn Japanese with Ki!

A week ago, I decided that I want to learn Japanese. According to my google research, the best way to learn any language is by watching enjoyable content. So, for the last 72 hours, I have been binge-watching "Attack on Titan", "My Neighbor Totoro", "Howl's Moving Castle", and "Queer Eye: We're in Japan!". These are the phrases I've learned so far!

Basics

  • Konnichiwa. Hello. こんにちは
  • Sayonara. Goodbye. さようなら
  • Hai. Yes. はい
  • Iie. No. いいえ
  • Arigato. Thank you. ありがとう
  • Gommenasai. I'm sorry. ごめんなさい

Extra

  • Kawaii. Cute. 可愛い
  • Yabai. Awesome/Awful. やばい
  • Ukeru. Haha. ウケる
  • Cho. Super. ちょ
  • Gaki. Brat. がき
  • Omurice. Omlet. オムレツ
